@ Genoix International Pvt Ltd.

Location: New Dehli, India

Android Developer

@ Banalogic

Location : Richmond Hill, Canada

This was my first co-op job. Coming into the job I had already mastered the basics of Java, android development, and possessed strong soft skills. Leaving the job I had mastered: Front-end and Back-end Android development using Java and C++; Crashlytics and deep mobile testing using Firebase; PHP token encryption authentication with UTC timestamp; Agile workflow; Git with Linux Command Line; Launching a beta-testing environment and publishing an app with legal parameters; Assembling a PC and modifying a Linux flavor to fit the developers needs. This Co-op was special to me because my supervisors were very kind and helpful. The environment they created encouraged me to work more than what was expected of me. I found the ticket system in Microsoft visual studio helped keep me competitive with the other co-op student to finish the most work. I would love to work here again because the wide variety of work kept me excited.

Software Developer + B.A

@ Carego Tek

Location: Burlington, Canada

I was very interested to join this job when I first started. As a software developer I was primarily in charge of fixing test cases for Telia, warehouse automation software, and developing solutions to new test cases. I also helped design screens for a website portal using C#. The bulk of my work was my position as business analyst. I participated in high level meetings to note and create solutions for my team, developed flow charts for the test cases, wrote up legal and technical documentation with coding structure, prepared weekly demos, and created visual designs for screens using photoshop. The workplace environment felt very formal and there was little room for creativity or expression of the mind.
